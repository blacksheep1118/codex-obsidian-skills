# License Policy

The repository uses the MIT License.

Each installable skill directory also contains a `LICENSE` file. This duplication is intentional: users may copy a single skill folder into `$CODEX_HOME/skills` without copying the repository root, so each copied folder should retain the license terms.

Do not add third-party source code, fixtures, decks, papers, or datasets unless their license allows redistribution in this repository. Prefer tiny synthetic fixtures for tests and examples.

When adding generated examples, make sure they do not reproduce private or copyrighted source material. A short illustrative Markdown sample is preferred over converted real courseware.
